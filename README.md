# what am I gonna eat website
Simple website to create weekly menus and generate a shoppinglist for all ingridents.

## Features
- Generate list of dishes between two weekdays
- Add dishes with tags and ingridents
- Generate shoppinglist for menu
- Individual dishes and menu for each user

## Tech Stack
Python, Flask, SQLAlchemy, SQLite, Bootstrap5, Flask-WTF forms

## Installation
```bash
pip install -r requirements.txt
python main.py
